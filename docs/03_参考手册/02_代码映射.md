# 代码结构映射

> **版本**: V1.8.12
> **更新日期**: 2026-02-26

本文档列出“应用接入”相关代码在本项目中的具体位置，便于快速定位前后端实现与数据库表结构。

## 1. 后端接口与路由

- **应用接入路由定义** (`/apps`，管理员鉴权)
  - [appRoutes.js](file:///c:/My_Project/account/routes/appRoutes.js)
  - 说明：所有 `/apps` 路由统一绑定 `requireAuth` 与 `requireAdmin`，包含 CRUD 接口。
- **路由挂载入口** (`/apps` 挂载在根路由)
  - [index.js](file:///c:/My_Project/account/routes/index.js)
  - 说明：`router.use("/apps", appRoutes)` 使 `/api/apps` 生效（Nginx 重写后为 `/apps`）。
- **应用接入控制器** (CRUD 逻辑)
  - [appController.js](file:///c:/My_Project/account/controllers/appController.js)
  - 说明：`createApp / getApps / getApp / updateApp / deleteApp` 对应应用的新增、查询、更新、删除。

## 2. SSO 与应用注册表

- **SSO 发放 Token 时从应用注册表匹配应用**
  - [ssoController.js](file:///c:/My_Project/account/controllers/ssoController.js)
  - 说明：根据 `public.applications` 表的 `allowed_origins` 匹配应用并选择 Token 生成策略。
- **内部微服务加载器** (Internal Micro-Apps)
  - [AppLoader.js](file:///c:/My_Project/account/src/core/AppLoader.js)
  - 说明：负责加载 `ai.apps.d/*.yaml` 配置的内部微服务。

## 3. 前端管理界面

- **管理后台入口与“应用接入”Tab**
  - [AdminPanel.tsx](file:///c:/My_Project/account/src/pages/AdminPanel.tsx)
  - 说明：左侧导航中包含“应用接入”入口，并加载 `AppManager` 组件。
- **应用接入管理组件** (列表/创建/编辑/删除)
  - [AppManager.tsx](file:///c:/My_Project/account/src/components/admin/AppManager.tsx)
  - 说明：应用列表、弹窗表单、密钥生成与复制逻辑均在此实现。
- **应用接入 API 服务层**
  - [appService.ts](file:///c:/My_Project/account/src/services/appService.ts)
  - 说明：封装 `/apps` 的 CRUD 调用，供 `AppManager` 使用。
