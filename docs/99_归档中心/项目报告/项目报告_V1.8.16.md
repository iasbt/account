## 3. 技术架构演进 (Architecture Evolution)

### 3.1 数据库变更 (Database Changes)
*   **Table: `oauth_codes`**:
    *   `code` (PK): 授权码
    *   `client_id`: 应用标识
    *   `user_id`: 用户标识
    *   `redirect_uri`: 回调地址
    *   `expires_at`: 过期时间
*   **Table: `applications`**:
    *   `allowed_origins`: `TEXT[]` 类型，支持多域名配置（如 localhost + 生产域名）。

### 3.2 部署与运维 (Deployment & Ops)
*   **自动化脚本**:
    *   `npm run migrate:oauth`: 自动创建 OAuth 相关表结构。
    *   `npm run migrate:gallery`: 自动应用相册服务数据库变更。
    *   `seed_apps.js`: 自动同步 `config/apps.js` 配置到数据库。
*   **Docker 适配**:
    *   优化了 Docker Compose 配置，确保 Postgres 容器健康检查通过后再启动后端服务。
