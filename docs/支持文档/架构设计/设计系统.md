# Account System Design System (Apple-Inspired)

> **设计哲学**: 高级、单色、内容优先  
> **参考**: Apple 官方设计风格 (2024)

## 1. 字体与排版 (San Francisco / System UI)

使用系统字体栈，保证跨设备原生渲染与一致性。

| 角色 | 尺寸 (桌面/移动) | 字重 | 字距 | 使用场景 |
| :--- | :--- | :--- | :--- | :--- |
| **Display** | 64px / 40px | Bold (700) | -0.022em | 大标题 |
| **Title 1** | 48px / 32px | Semibold (600) | -0.015em | 页面标题 |
| **Title 2** | 32px / 24px | Medium (500) | -0.010em | 区块标题 |
| **Body** | 17px / 16px | Regular (400) | Normal | 正文内容 |
| **Caption** | 14px / 13px | Regular (400) | Normal | 提示与说明 |

## 2. 颜色体系 (Monochrome)

整体遵循灰阶主调，配合语义化强调色。

### 背景
- **Base**: `#f5f5f7`（浅灰）- 页面主背景  
- **Surface**: `#ffffff`（纯白）- 卡片与弹窗  
- **Contrast**: `#000000`（纯黑）- 底部与大块背景  

### 文字
- **Primary**: `#1d1d1f`（深灰黑）- 标题与正文  
- **Secondary**: `#86868b`（中性灰）- 说明与副标题  
- **Tertiary**: `#d2d2d7`（浅灰）- 占位与弱提示  

### 强调色
- **Action Blue**: `#0071e3`（主行动色）- 链接、主按钮  
- **Destructive**: `#ff3b30`（危险色）- 删除、错误  
- **Success**: `#34c759`（成功色）- 成功提示  

## 3. 布局与间距

### 栅格
- **容器宽度**: 最大 `1280px`  
- **页面边距**: `24px`（移动）/ `40px`（平板）/ `80px`（桌面）  
- **栅格间距**: `24px` 或 `32px`  

### 卡片
- **圆角**: `20px`（标准），`30px`（大卡片）  
- **内边距**: `40px`  
- **阴影**: `0 4px 6px -1px rgba(0,0,0,0.02)`  

## 4. 组件规范

### 按钮
- **主按钮**: 胶囊形（`rounded-full`），`bg-[#0071e3]`，`text-white`  
- **次按钮**: 链接样式，`text-[#0071e3]`，`hover:underline`  
- **弱按钮**: 边框或浅色背景  

### 输入框
- **风格**: 极简、大内边距  
- **边框**: `#d2d2d7`（默认），`#0071e3`（聚焦）  
- **圆角**: `12px`  

## 5. 动效
- **时长**: `300ms`（标准）  
- **缓动**: `cubic-bezier(0.25, 0.1, 0.25, 1.0)`  
- **悬浮**: 缩放 `1.02`（轻微提升）
