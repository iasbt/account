# 数据库架构文档 (Database Architecture)

> **版本**: V1.8.6
> **生效日期**: 2026-02-25
> **状态**: Active

## 1. 概述

Account 系统作为中心化身份验证服务，管理着用户身份、应用接入注册表以及邮件服务配置。数据库采用 **PostgreSQL 14**。

## 2. 模式 (Schemas)

*   `public`: 核心账户、应用注册表、邮件服务。
*   `gallery`: 图库子应用数据 (独立 Schema)。

## 3. 核心表结构 (Public Schema)

### 3.1 用户表 (`users`)
*注意：此表为 V1.8.6 启用的标准用户表，使用 UUID (Text) 作为主键，替代了旧的 `legacy_users`。*

| 字段 | 类型 | 说明 | 约束 |
| :--- | :--- | :--- | :--- |
| `id` | `TEXT` | 用户 ID (UUID) | PK |
| `name` | `VARCHAR(255)` | 用户名 | NOT NULL |
| `email` | `VARCHAR(255)` | 邮箱地址 | UNIQUE, NOT NULL |
| `password` | `VARCHAR(255)` | Bcrypt 哈希密码 | NOT NULL |
| `avatar_url` | `VARCHAR(255)` | 头像链接 | |
| `is_admin` | `BOOLEAN` | 是否管理员 | DEFAULT FALSE |
| `created_at` | `TIMESTAMP` | 创建时间 | DEFAULT NOW() |
| `updated_at` | `TIMESTAMP` | 更新时间 | DEFAULT NOW() |

### 3.2 遗留用户表 (`legacy_users`)
> **DEPRECATED (已废弃)**: 数据已全量迁移至 `users` 表，此表仅作归档保留，即将删除。

| 字段 | 类型 | 说明 | 约束 |
| :--- | :--- | :--- | :--- |
| `id` | `INTEGER` | 用户 ID | PK |
| `email` | `VARCHAR` | 邮箱 | UNIQUE |
| `password_hash` | `VARCHAR` | 密码哈希 | |
| `is_admin` | `BOOLEAN` | 是否管理员 | DEFAULT FALSE |
| `created_at` | `TIMESTAMP` | 创建时间 | |

### 3.3 应用注册表 (`applications`)
*用于管理接入 Account SSO 的子应用。*

| 字段 | 类型 | 说明 | 约束 |
| :--- | :--- | :--- | :--- |
| `id` | `UUID` | 内部 ID | PK, DEFAULT gen_random_uuid() |
| `app_id` | `VARCHAR` | 应用唯一标识 (如 `gallery`) | UNIQUE, NOT NULL |
| `name` | `VARCHAR` | 应用显示名称 | NOT NULL |
| `allowed_origins` | `TEXT[]` | 允许的回调域名列表 | NOT NULL |
| `token_type` | `VARCHAR` | 令牌类型 (`supabase`, `standard`) | NOT NULL |
| `secret` | `VARCHAR` | 应用专属签名密钥 | NOT NULL |
| `is_active` | `BOOLEAN` | 启用状态 | DEFAULT TRUE |
| `created_at` | `TIMESTAMP` | 创建时间 | DEFAULT NOW() |

### 3.4 邮件服务提供商 (`email_providers`)
*支持多 SMTP 服务商配置。*

| 字段 | 类型 | 说明 | 约束 |
| :--- | :--- | :--- | :--- |
| `id` | `SERIAL` | ID | PK |
| `name` | `VARCHAR` | 提供商名称 (如 `Outlook`) | NOT NULL |
| `host` | `VARCHAR` | SMTP 主机 | NOT NULL |
| `port` | `INTEGER` | SMTP 端口 | NOT NULL |
| `secure` | `BOOLEAN` | 是否使用 SSL/TLS | DEFAULT TRUE |
| `auth_user` | `VARCHAR` | 认证用户名 | NOT NULL |
| `auth_pass_encrypted`| `VARCHAR` | 加密后的密码 | NOT NULL |
| `from_name` | `VARCHAR` | 发件人显示名称 | |
| `from_email` | `VARCHAR` | 发件人邮箱 | |
| `is_active` | `BOOLEAN` | 是否启用 | DEFAULT TRUE |
| `created_at` | `TIMESTAMP` | 创建时间 | DEFAULT NOW() |

### 3.5 邮件模板 (`email_templates`)
*存储动态邮件内容模板。*

| 字段 | 类型 | 说明 | 约束 |
| :--- | :--- | :--- | :--- |
| `type` | `VARCHAR` | 模板类型 (如 `verification_code`) | PK |
| `subject` | `VARCHAR` | 邮件标题模板 | NOT NULL |
| `content` | `TEXT` | HTML 内容模板 | NOT NULL |
| `variables` | `JSONB` | 可用变量描述 | |
| `updated_at` | `TIMESTAMP` | 最后更新时间 | DEFAULT NOW() |

### 3.6 验证码 (`verification_codes`)

| 字段 | 类型 | 说明 | 约束 |
| :--- | :--- | :--- | :--- |
| `id` | `BIGSERIAL` | ID | PK |
| `email` | `TEXT` | 接收邮箱 | NOT NULL |
| `code` | `TEXT` | 验证码 | NOT NULL |
| `expires_at` | `TIMESTAMP` | 过期时间 | NOT NULL |
| `is_used` | `BOOLEAN` | 是否已使用 | DEFAULT FALSE |
| `created_at` | `TIMESTAMP` | 创建时间 | DEFAULT NOW() |

## 4. 图库 Schema (`gallery`)

### 4.1 图片表 (`images`)

| 字段 | 类型 | 说明 | 约束 |
| :--- | :--- | :--- | :--- |
| `id` | `SERIAL` | ID | PK |
| `title` | `VARCHAR` | 图片标题 | |
| `file_path` | `VARCHAR` | 存储路径 | NOT NULL |
| `created_at` | `TIMESTAMP` | 上传时间 | DEFAULT NOW() |

## 5. 索引策略

*   `users(email)`: 唯一索引，加速登录查询。
*   `applications(app_id)`: 唯一索引，加速 SSO 应用匹配。
*   `verification_codes(email, code)`: 复合索引，加速验证码校验。
