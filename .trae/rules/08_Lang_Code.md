# Language Standard (语言规范) - 代码与工具篇

> **Status**: Active
> **Effective Date**: 2026-02-22
> **Enforcement**: 🛡️ **Strict (强制执行)**
> **Language**: Simplified Chinese (简体中文)

## 1. 代码与路径
*   **例外**: 代码变量名、文件路径、API 路由、Docker 容器名、命令行指令等必须保持 **英文**。
*   **引用**: 在文档中引用代码实体时，应直接使用其英文名称。

## 2. 技能例外
*   **技能同步**: `.trae/skills/**/SKILL.md` 允许保留上游英文内容，用于确保技能与官方版本一致。

## 3. 自动生成报告
*   **要求**: 所有工具生成的审计报告、验证报告，其描述性文字必须翻译为中文输出。
