# SSO 阶段 1（基础一致性）

## 目标
- 登录回跳来源优先
- 非法回跳拦截
- 回跳路径策略统一

## 改造范围
- Account 登录页回跳逻辑
- 登录失败与回跳错误提示

## 技术选型
- React Router 现有状态传递
- 现有 allowlist 白名单机制

## 风险评估
- 低：仅影响登录回跳逻辑

## 回滚方案
- 还原为仅使用 redirect 参数或默认回首页

## 验收标准
- 通过 /login 登录后优先回到来源路径
- 非法 redirect 被阻断且不跳转

## 进度
- 状态: 已完成
- 已完成: 登录回跳来源优先、非法回跳拦截、回跳路径策略统一、单元测试
